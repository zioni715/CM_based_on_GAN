# CM_based_on_GAN


# GAN 기반 합성 데이터를 활용한 건설 현장 안전장비 미착용 탐지 성능 향상

> **Status:** 🧪 Planning / 기획 단계  
> **Keywords:** GAN, Synthetic Data, PPE Detection, Construction Safety, YOLOv8

---

## 1. 프로젝트 개요

이 프로젝트는 **건설 현장 작업자의 안전장비(PPE; Personal Protective Equipment) 미착용을 자동으로 탐지**하는 AI 모델을 구축하고,  
여기에 **GAN(Generative Adversarial Network)** 으로 생성한 **합성 위험 상황 이미지(Synthetic Data)** 를 추가 학습시켜  
탐지 성능을 향상시키는 것을 목표로 한다.

특히 현실 현장에서는 **안전모, 안전조끼, 안전화 미착용 장면**이 자주 발생하지 않거나,  
의도적으로 촬영하기 어려워 **데이터가 극도로 부족한 문제**가 존재한다.  
본 프로젝트는 이를 “GAN 기반 데이터 생성”으로 보완하여,  
**희귀·위험 상황에 대한 일반화 성능을 개선**하는 방향을 탐색한다.

---

## 2. 연구 배경 및 필요성

- 건설 산업은 여전히 **추락, 낙하물, 협착** 등 중대 재해가 빈번하게 발생하는 고위험 산업 분야이다.
- 안전사고 예방을 위해 **CCTV, 드론, 360° 카메라** 등의 영상 데이터를 활용한  
  **AI 기반 실시간 안전 모니터링 시스템** 도입이 점차 확대되고 있다.
- 그러나 실제 데이터에는 다음과 같은 한계가 있다:
  - **PPE 미착용 장면이 적음** → 클래스 불균형 심각
  - **실제 사고·위험 상황 촬영은 윤리·법적 제약** 존재
  - 다양한 작업 환경(야간, 역광, 먼 거리, 가림, 소규모 인원 등)을 모두 커버하기 어렵다.

이러한 이유로, 기존 객체 탐지 모델(YOLO 시리즈 등)을 그대로 적용하면  
**정작 중요한 “미착용/위험 상황”에서 탐지 성능이 급격히 떨어지는 문제가 발생**한다.  

따라서, **실제 데이터를 보완할 수 있는 합성 데이터(Synthetic Data)** 가 필요하며,  
그 중에서도 **GAN은 “현실과 유사한 이미지”를 생성할 수 있는 강력한 도구**로 주목받고 있다.

---

## 3. 프로젝트 목표

이 프로젝트의 최종 목표는 다음과 같다:

1. **PPE 객체 탐지 모델 구축**
   - 대상 PPE:
     - 👷‍♂️ 안전모(Helmet)
     - 🦺 안전조끼(Vest)
     - 🥾 안전화(Safety Shoes)
   - 사람(Person)과 PPE 객체를 인식한 후,
     **“사람-장비 매칭”을 통해 미착용 여부를 자동 판단**하는 로직 설계

2. **GAN 기반 합성 데이터 생성 파이프라인 설계**
   - 실제 현장 이미지 또는 유사 데이터에서
     - “PPE 착용 상태” → **“PPE 미착용 상태”로 변환**하는 합성 이미지 생성
     - 예: 헬멧이 있는 영역을 인페인팅(Inpainting)하여 자연스러운 “무헬멧” 이미지 생성
   - 동일 배경·동일 인물·동일 자세에서 **PPE 유/무만 달라지는 데이터셋** 확보

3. **합성 데이터 활용에 따른 성능 향상 검증**
   - Baseline:
     - 실제 데이터만으로 학습한 PPE 미착용 탐지 모델
   - Proposed:
     - 실제 데이터 + GAN 합성 데이터를 함께 학습한 모델
   - 두 실험을 비교하여:
     - 특히 **“미착용 상황”에서의 Recall / F1-score 향상 여부**를 정량적으로 평가

---

## 4. 기대 효과

- **데이터 희소성 문제 완화**  
  실제로 수집하기 어려운 PPE 미착용 장면을 GAN으로 생성함으로써,  
  위험 상황 데이터 부족 문제를 완화할 수 있다.

- **희귀·위험 상황에 대한 일반화 성능 향상**  
  다양한 각도, 조명, 배경에서의 “미착용 이미지”를 충분히 공급함으로써,  
  실제 현장에서의 **탐지 민감도(Recall) 향상**이 기대된다.

- **스마트 건설 안전관리 시스템의 핵심 모듈로 확장 가능**  
  - CCTV 기반 실시간 모니터링
  - 디지털트윈/현장 대시보드와 연계한 **안전 상태 시각화**
  - 건설사/발주처의 **안전관리 고도화 솔루션**의 기반 기술로 활용 가능
